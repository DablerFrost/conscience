Scoring Methodology
CONSCIENCE scores are computed on a 0–100 scale using four weighted data sources. Scores reflect point-in-time analysis of publicly available disclosures. Updated as new CDP cycles, InfluenceMap assessments, or material regulatory events occur. All source data is linked per company card.

Open Proof Protocol: find an error in any score or data point and it will be published as a public correction. Contact: contact@vyrion.earth · Methodology: VYRION-METHODOLOGY-v1.0.md
